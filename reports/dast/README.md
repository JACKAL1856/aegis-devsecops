# OWASP ZAP DAST Report

This HTML report was generated using OWASP ZAP during Phase 4 of the Aegis DevSecOps pipeline.

## Contents
- Application crawl results
- Vulnerability findings
- CWE and WASC mappings
- Evidence and affected URLs

## Usage
Download `dast-report.html` and open it in a browser for full interactive analysis.

## Note
This report was generated against a test application in a lab environment for learning purposes.
